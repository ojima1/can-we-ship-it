# Can We Ship It?

### AI writes code. Engineers decide if it ships.

Reviewing AI-generated .NET code and turning it into production-ready software.

🎥 YouTube Series • 💻 GitHub • ✍🏾 LinkedIn


# About

Can We Ship It? is a hands-on engineering series where every episode starts with an AI-generated Pull Request.

Instead of blindly accepting the code, we review it like a real software engineering team.

Every Pull Request answers one question:

🚢 Can We Ship It?

# SHIP Framework™

Every review is scored using the SHIP Framework.

| Category       | Description                               |
|--------------- |-------------------------------------------|
| Security       | Is the code secure?                       |
| Health         | Is it correct, reliable and maintainable? |
| Implementation | Is the architecture clean and scalable?   |
| Performance    | Will it perform well in production?       |

Every category is scored out of 25 points.

Maximum score:

100
